# Image assets structure

Images are organized by **page** and **section**. Paths are relative to `public/`; use in code as `/images/...`.

## Home page (`/images/home/`)

| File | Section / usage |
|------|------------------|
| `logo.svg` | Header, layout favicon |
| `overview_badge.svg` | OverviewCardsSection |
| `line_vr.svg` | VRTourSection |
| `footer_phone.svg`, `footer_mail.svg`, `footer_back_to_top.svg` | Footer |
| `vision_dot.svg`, `vision_click.svg`, `vision_icon.svg`, `vision_next_action.svg`, `vision_bg_mb.svg` | VisionMissionSection |
